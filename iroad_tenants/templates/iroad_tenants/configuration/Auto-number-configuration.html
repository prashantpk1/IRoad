    Organization Profile Auto Numbering

    Manage auto numbering settings and formats (currently enabled for Organization Profile).

    {% csrf_token %}

    Numbering Rules

    The number the system will assign to the next item.
    Set the number of digits for the serial number. If the number is shorter than this length, zeros will be added to the left. Example: if the length is 5 and the current number is 3, it will appear as 00003.

    Format & Uniqueness

    Select the format to use for generating numbers (digits, letters, or combinations).
    Ensure that each number in the sequence is unique and not duplicated.
