Service Item Settings

Configure margin requirements and formatting rules for service items.

Margin Settings

Configure minimum margin constraints and override policies.

Margin Rules

When enabled, selling prices must meet or exceed the predefined minimum margin percentage.
When enabled, authorized users can accept margins below the minimum requirement.
When enabled, overriding the minimum margin requires providing a justification.

Default Values

Choose whether the default margin is a percentage or a fixed amount.
If Margin Type is Percentage, value is in %; if Fixed Amount, value is in base currency.
Defines whether the margin is calculated from the sell price or the cost price.
