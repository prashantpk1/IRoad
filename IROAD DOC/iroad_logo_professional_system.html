Usage Guidelines

Best practices for consistent brand application

✓ DO

  • Use Dynamic Blue (#3B82F6) on light backgrounds
  • Use White (#FFFFFF) on dark backgrounds
  • Maintain proper spacing around logo
  • Scale proportionally to all sizes
  • Keep typography clean and readable
  • Use supporting colors for UI elements
  • Maintain color consistency across materials
  • Test contrast on different backgrounds

✕ DON'T

  • Use Primary Blue (#1E40AF) for logo text
  • Use Speed Orange (#F59E0B) for logo text
  • Use other colors for logo - only Dynamic Blue or White
  • Distort or transform the logo text
  • Add drop shadows or effects to logo
  • Mix logo colors with non-approved shades
  • Use on low-contrast backgrounds
  • Reduce size below minimum specifications
